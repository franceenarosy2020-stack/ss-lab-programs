# System Software Lab Programs

## Overview

This repository contains System Software lab programs completed as part of the B.Tech Computer Science and Engineering curriculum (KTU).

The programs are written in C and cover fundamental concepts of system software including assemblers, loaders, linking, scheduling algorithms, file organization methods, and resource allocation techniques.

## Topics Covered

- Two-Pass Assembler
- Loaders and Linking
- Sequential File Organization
- Indexed File Organization
- CPU Scheduling
- Resource Allocation
- System Software Fundamentals

## Repository Contents

The repository contains programs related to:

- Pass 1 of Two-Pass Assembler
- Pass 2 of Two-Pass Assembler
- Linking Loader Concepts
- Sequential File Organization
- Indexed File Organization
- CPU Scheduling
- Banker's Algorithm
- Supporting input and output files used in the experiments

## How to Run

Compile a program:

```bash
gcc FileName.c -o output
```

Run the program:

```bash
./output
```

## Purpose

These programs were developed as part of System Software laboratory work to understand the design and implementation of assemblers, loaders, file organization techniques, scheduling algorithms, and other core system software concepts.
